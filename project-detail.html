Overview

A web-based library management system designed to digitize and simplify library operations including book inventory, member registration, and borrow/return tracking.

Technologies Used
MongoDB Express.js React.js Node.js JavaScript CSS3 HTML5
Key Features
  • Book inventory management
  • Member registration and management
  • Borrow and return tracking
  • Search and filter functionality
  • Responsive design for all devices
  • Admin dashboard
