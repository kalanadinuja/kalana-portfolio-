Featured Project

SMART CAMPUS
WEB APP

A comprehensive campus management web application built with Spring Boot that enables universities to manage students, faculty, facilities, and campus services through a centralized digital platform.

Live Demo View on GitHub
Overview

Smart Campus is a web-based university management platform built with Spring Boot that digitalizes campus operations including student management, facility booking, course management, and campus announcements.

Technologies Used
Spring Boot Java MySQL Thymeleaf HTML5 CSS3 JavaScript Maven
Key Features
  • Student and faculty management
  • Course and timetable management
  • Campus facility booking system
  • Announcements and notice board
  • Admin dashboard
  • Responsive web interface
My Role

I designed and developed the entire web application including backend API development, database design, and frontend implementation.

  • Full Stack Development
  • Backend Development with Spring Boot
  • Database Design
  • UI/UX Design

Challenges
  • Designing a scalable backend architecture for multiple campus modules
  • Managing complex relationships between students, courses, and facilities in MySQL
What I Learned
  • Improved Spring Boot and Java backend development skills
